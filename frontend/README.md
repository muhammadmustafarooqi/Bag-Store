# KAARVAN Frontend 🛍️

The premium frontend for the KAARVAN bag store, built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

---

## 🚀 Features

- **Luxury Design**: High-end fashion aesthetic with gold accents and minimalist layouts.
- **Secure Auth**: Full flow for email verification, password resets, and Google Login.
- **Dynamic Shop**: Advanced filtering, search, and category-based navigation.
- **Smooth UX**: Framer Motion animations and responsive design for all devices.
- **Localized**: Full support for Pakistani regions and phone formats.

---

## ⚡ Getting Started

### 1. Install Dependencies
```bash
npm install
```

### 2. Environment Setup
Create a `.env.local` file:
```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api/v1
NEXT_PUBLIC_WHATSAPP_NUMBER=923001234567
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_id_here
```

### 3. Run Development Server
```bash
npm run dev
```

## 📂 Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Data Fetching**: React Query (TanStack)
- **Auth**: Google OAuth + JWT
- **Animations**: Framer Motion
